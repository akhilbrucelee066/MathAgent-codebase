About the App

This is an agentic mathematics assistant. Enter any maths query to get step-by-step solutions, explanations, or concepts.


LLM: llama-3.3-70b-versatile
RAG: MathQA dataset from Hugging Face
Web Search: Serper web search API
Tech Stack: FastAPI, HTML, CSS, JS

The agent uses a knowledge base and web search to provide the best possible answer, and you can give feedback on every answer to help improve the system.
